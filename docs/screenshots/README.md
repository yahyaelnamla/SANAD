# Screenshots

No marketing screenshots are checked into the repository (binary assets are excluded to keep the clone lean).

## Recommended captures for judges

After starting the stack (`.\scripts\start-sanad.ps1`), capture:

| Page | URL | What to show |
|------|-----|--------------|
| Landing | http://localhost:3000/welcome | Hero, partner logos, demo questions |
| Chat | http://localhost:3000/chat | Answer with execution trace expanded |
| Evaluation | http://localhost:3000/evaluation | Judge harness dashboard |
| Scanner | http://localhost:3000/scanner/company | AAOIFI screening result |

Save as `docs/screenshots/welcome.png`, `chat.png`, `evaluation.png`, etc., and link them from the root README if you add them before submission.

Icons already in repo: `frontend/public/icons/` (Fanar, QCRI, HBKU, PWA).
